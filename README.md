# MAT – Micro Anonymous Twitter
Micro Twitter that can be easily deployed on AWS using Terraform

# Project Structure

Here is the files tree of the project:

```
.
├── app
│   ├── Dockerfile
│   ├── app.py
│   └── templates
│       └── template.html
└── terraform
    ├── config.tf
    ├── data.tf
    ├── main.tf
    ├── outputs.tf
    └── variables.tf
```
